# Startup Analysis Memo Template

Use this template structure for the full memo output. Each section has formatting guidance — scale the depth to the available data. A good memo is 1,500-2,500 words total.

---

```markdown

**Sector:** [Sector Name]
**Stage:** [Series X] | **Valuation:** [$XB] | **Total Raised:** [$XM]
**Composite Score:** X.X / 5.0 — [Verdict]

---
```
